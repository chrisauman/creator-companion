Marketing hero — slow blue wave variants

Eight directions all built around slow-moving brand-cyan waves. Same headline / sub / CTA in each so you can compare animation against identical content. All cycle in the 12–40 second range so they read as ambient, not busy. Respects prefers-reduced-motion.
